Delete a book

  1. Open the collection that has the book (book, calendar, and so on) that you want to delete.

  2. In the Collections tab, right-click the book that you want to delete, and then click  Delete Book.

The Confirm Delete dialog box appears.

    • To move that book to the Recycle Bin, click Delete. Otherwise, click Cancel.
