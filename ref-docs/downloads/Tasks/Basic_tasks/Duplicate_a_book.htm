Duplicate (copy) a book

  1. Open the collection that has the book you want to duplicate.

  2. In the Collections tab, do one of these steps:

    • Right-click the book, and then click Duplicate Book.

    • Click the book you want to duplicate. Click the menu button () that appears next to that book thumbnail, and then click Duplicate Book.

A duplicate of the book is added to the collection. The word Copy is appended to the folder name. You can see the folder name if you use Show in File Explorer (more).
