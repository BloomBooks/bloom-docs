Delete or rename a collection

A folder named Bloom is added to your computer when you install the Bloom program. It contains one or more subfolders. Each of those subfolders is a collection.

Delete a collection

  1. In the Collections tab, right-click a book and then click Show in File Explorer.

  2. Leave the computer file browser open, but close Bloom.

  3. Go to the Bloom folder:

Windows® - My Documents\Bloom or Documents\Bloom

Linux - /home/<user name>/Bloom.

  1. Find the subfolder that has the name of the collection that you want to delete.

  2. Delete that subfolder.
