Choose settings protections

Bloom supports roles. Sometimes, a person who should not change settings will do it anyway. You can do two things to stop this.

  1. Open the Settings dialog box.

  2. Click Settings Protection.
    The Settings Protection dialog box opens.

  1. Select () or clear () one or both of these check boxes:

    • Hide the button that opens settings
      If it is selected, fewer buttons appear on the toolbars.

    • Require the factory password to get into settings
      If it is selected, a password is needed. Otherwise, you cannot open the Settings dialog box or the Open/Create Collection dialog box.

  1. Click OK.

Tip

  • The lock symbols you see are described here.

  • To see the buttons that are hidden, press Ctrl+Shift. You can click them while you see them.

  • The factory password appears in the Settings Protection dialog box.
