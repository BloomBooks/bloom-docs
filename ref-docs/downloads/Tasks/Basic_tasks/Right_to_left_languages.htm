Right-to-left languages

You can use Bloom to make books that use right-to-left languages. You need to set up Bloom and also set up a keyboard so you can type the characters.

Set up Bloom

Do these steps in the Book Making tab:

  1. Choose the default font that can display the characters for the language.

  2. Select () the Right to Left Writing System check box for the left to right language.
