Choose a text color for a style

Use this dialog box to choose the text color for the current style. 

  1. Open the Format dialog box and then click the Characters tab.

  2. In the Characters tab, click the Color control.

  3. Use the controls in the Text Color dialog box.

  4. Click OK.

Note

  • Apply direct formatting is different.

  • The Background Color dialog box is similar.
