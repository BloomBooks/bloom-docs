Choose or create a style

On some pages, you can choose styles or create new styles.

  1. In the text box that you want to change, click the gear icon ().

The Format dialog box opens.

  1. On the Style Name tab, click the Style box, and then click the style you want to use.

If you do not see the style you need, do these steps:

    • Click the Create a new style link.

    • Type a name for the new style in the New Style box, and then click Create.

The new style is the current (selected) style so you can configure it.

Important

  • Not all the styles used in the current book appear in the Style box drop-down list.
    For example, some styles are only used on cover pages, but not inside pages. Also, some styles like BigWords are used in Picture & Word pages, but not other inside pages.

  • Style names you add can only have alphabetical characters and numbers. You cannot use spaces, hyphens, and so on.

  • When you do this task, you may want to open a book in the collection where it was created (where it was an original book).
