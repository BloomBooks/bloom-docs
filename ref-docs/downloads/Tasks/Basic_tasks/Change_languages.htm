Change or Set languages

  1. Open the Settings dialog box.

  2. Click the Languages tab.

  3. Click the Change hyperlink or the Set hyperlink that is below the language you will change or set (add).

    • Use the Choose Language dialog box to search for and select the language. You can also edit the language name to change how it is displayed.

If necessary, if you clicked Change, and a language was chosen, you can also click  CUSTOMIZE to open the Custom Language Tag dialog box. Choose the script, country where it is used, and enter any variant (dialect). The BCP 47 Tag will show the language and also that script, country and variant. Click OK.

If necessary, if you clicked Set or have not yet chosen a language, the CREATE UNLISTED LANGUAGE (qaa-x-) control appears. You can click it and then enter the name and country in the Unlisted Language Tag dialog box. The BCP 47 Tag will show qaa and that name and country. Click OK.

  • Click OK.

  2. Language 3 and Sign Language can show the Remove hyperlink. You can click it to remove the language (optional).

  3. Click OK.

Important

Here are additional things you can do:

  • Use the Edit tab toolbar to select one or more languages to display in text boxes.

  • Use the Language tab in the Text Box Properties dialog box to select a language for a particular text box.

  • Select the languages you want to include when you publish a book to the web.

  • Set a sign language when you publish your sign language book to the web. It appears with Features.
