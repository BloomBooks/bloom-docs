Get a book from BloomLibrary.org

BloomLibrary.org has books that others have published to the web. You can download a book and make a book from it. If it has audio, that audio is also downloaded as part of the shell book.

Prerequisites

  • You need to have an Internet connection.

Do these steps:

  1. Click the Collections tab.

  2. Click the link called GET MORE SOURCE BOOKS FROM BLOOMLIBRARY.ORG.

    The Internet site opens to https://bloomlibrary.org/read.

  3. Use the search box to find a book.

  4. You can read it.

If the author allowed these options, you can also:

  • Download it into Bloom.

  • Download it as a PDF, EPUB or BloomPUB file.

Note

  • Here are videos to help you learn about downloading and then using shell books:
