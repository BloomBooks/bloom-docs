Enter project information

  1. Open the Settings dialog box.

  2. Click the Project Information tab.

Type or edit names in any of these boxes: Country, Province, District, or Bloom Collection Name.

  1. Click OK.

Tip

  • You might also want to enter your Subscription Code while the Settings dialog box is open.

  • Continue with the steps in Create a Bloom collection if that is what you are doing.
