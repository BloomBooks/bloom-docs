Change User Interface language

  1. In any of the tabs, click the UI language control on the toolbar and then click the language you want in the drop-down list.

  2. If the language you want is not listed, do any of these options:

    • Select () Show translations which have not been approved yet if you need to see or use an unapproved UI language.

    • Click Help us Translate Bloom to open https://crowdin.com/project/sil-bloom in the Internet.

    • Get more help.
