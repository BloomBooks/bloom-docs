Make a book from Big Book template

  1. Click Collections tab.

  2. Below Sources for New Books, click the Big Book template.

    The cover page of the template appears in the main pane. You also see information about the template.

  3. Click MAKE A BOOK USING THIS SOURCE.

    The book is added to your collection. It opens in the Edit tab. You see the Front Cover page.

  4. On the toolbar, select () languages.

  5. On the Front Cover page, do these steps:

    • Type the book title in the text box (near the top of the page) in the local language.

    • Type the title in the next text box using language 2.

    • Choose a picture.

    • Type information about the author/illustrator or anything else.

    • Choose a topic for the big book.

  1. In the Pages pane, click Inside Front Cover and then type any information you want to appear on that side of the cover.

  2. In the Pages (left) pane, click Title Page:

    • Edit a title, if necessary.

    • Type any contribution information made by writers, illustrators, editors and other people.

    • Type words that acknowledge those who helped pay for this book.

    • Review the local language and project information near the bottom of the page.

  3. In the Pages pane, click Credits Page:

    • Use the Copyright and License dialog box.

    • Type the name of translator.

    • Type words of acknowledgement.

    • Type the ISBN in the ISBN box (if your organization uses them).

  4. In the Pages pane, click Instructions page.

    • Read the instructions. You can translate and replace the instructions (optional).

  5. Add a page and then do any of these steps:

    • Type words in the text boxes that are displayed on the page.

    • If the page can hold a picture, choose the picture.

    • Repeat this step for each page.

  6. Type any additional information in the back cover pages.

  7. On the Title Page, click the Paste Image Credits link again to update the image credits.

Tip

  • Create a shell book for others to use.

  • Formatting Text overview help you work with styles and formatting.

  • In the Collections tab, if you click the big book you made, you can see information about big books. You see it in the right pane below the cover page.
