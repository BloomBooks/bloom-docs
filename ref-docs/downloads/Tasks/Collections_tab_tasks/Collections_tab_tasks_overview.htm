Collections tab tasks overview

When the Collections tab is displayed, you can see these links below Books From BloomLibrary.org:

  • GET MORE SOURCE BOOKS AT BLOOMLIBRARY.ORG

Click this link to open https://bloomlibrary.org/ so you can get a book.

  • SHOW ANOTHER COLLECTION

Click this link to open the Bloom folder on your computer. If there are other collection folders there, do these steps: double-click to open that folder, click the *.bloomCollection file, and then click Open. 

The books in that collection appear in Bloom.

  • Click  near the collection name if you do not want to see this collection in Bloom.

Do any of these tasks when in the Collections tab is displayed:

  • Make a book from Arithmetic template

  • Make a book from Basic Book template

  • Make a book from Big Book template

  • Make a book from a downloaded book

  • Make a book from a sample shell

  • Make a Wall Calendar

  • See Also: Collection tab commands

Note

  • Consider using any of the tools in the tool box.
