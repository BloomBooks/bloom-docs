Make a book from a downloaded book

You can get a book from BloomLibrary.org. Then, you can use it to make a translated book for your collection. It is expected that you will use the pictures and pages as set in the source book. You may not need to do all the steps in this task.

  1. Click the Collections tab.

    • In the left pane, look below Books From BloomLibrary.org.

  2. Click the downloaded book you want to use to make a new book.

  3. Click MAKE A BOOK USING THIS SOURCE.

    The book is added to your collection. It opens in the Edit tab. You see the Front Cover page.

  4. On the toolbar, select () languages if the book will use two or three languages.

  5. At the top of the Front Cover page, type the title of the book in the local language.

  6. In the Pages pane, click Inside Front Cover and then type information you want to appear on this side of the cover.

  7. In the Pages pane, click Title Page:

    • Type any contribution information made by writers, illustrators, editors and other people.

    • Type words that acknowledge any funding agency that helped pay for this book.

    • Review the local language and project information at the bottom of the page.

  8. In the Pages pane, click Credits Page:

    • Use the Copyright and License dialog box to update those metadata for the translated book.

    • Type the name of the translator.

    • Type a translation of the acknowledgements in language 2.

    • Review the ISBN in the ISBN box (if your organization uses them).

  9. In the Pages pane, click the next page:

    • Click the text box that will store the local language text.

    • Read the source text.

    • Type a translation of that source text in the local language.

    • If this collection uses a language 3, then type the translation into that language too.

    • Repeat this step for each page.

  10. Type any additional information in the Back Cover pages.

  11. On the Title Page, click the Paste Image Credits link to update the image information.

  12. If the book has a Translation Instructions page, follow those instructions.

Tip

  • Formatting Text overview help you work with styles and formatting.

  • It is expected that you will use the pictures that are in the book that you downloaded.
