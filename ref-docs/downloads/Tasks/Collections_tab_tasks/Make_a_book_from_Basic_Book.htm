Make a book from Basic Book template

Here, the steps are for the Basic Book template, but the steps for other available templates are similar. See Also: Is this an update of your existing book?

  1. Click Collections tab.

  2. Below Sources for New Books, click the Basic Book template.

    The cover page of the template appears in the main pane.

  3. Click MAKE A BOOK USING THIS SOURCE.

    The book is added, and opens in the Edit tab. You see the Front Cover page.

  4. On the toolbar, do any of these steps:

    • Select () languages.

    • Choose a page size and orientation.

  5. On the Front Cover page, do these steps:

    • Type the book title in the text box (near the top of the page) in the local language.

    • Type the title in the text box using language 2.

    • Choose a picture.

    • Type the name of the author/illustrator or any other information in the text box below the picture.

    • Choose a topic.

  6. In the Pages pane, click Inside Front Cover and type any information you want to appear on that side of the cover.

  7. In the Pages pane, click Title Page:

    • Edit a title, if necessary.

    • Type any contribution information made by writers, illustrators, editors and other people.

    • Click the bottom text box and then type words that acknowledge those who helped pay for this book.

    • Review the local language and project information near the bottom of the page.

  8. In the Pages pane, click Credits Page:

    • Use the Copyright and License dialog box.

    • Type the name of the translator.

    • Type words of acknowledgement.

    • Type the ISBN in the ISBN box (if your organization uses them).

  9. Add a page and then do any of these steps:

    • Use the Change Layout control (optional).

    • Type words in the text boxes that are displayed on the page.

    • If the page can hold a picture, add a picture.

    • Repeat this step for each page.

  10. Type any additional information in the back cover pages.

  11. On the Title Page, click the Paste Image Credits link again to update the image credits.
