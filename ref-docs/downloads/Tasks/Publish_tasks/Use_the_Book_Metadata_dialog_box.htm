Use the Book Metadata dialog box

  1. Click ePUB, and then click Book Metadata.

The Book Metadata dialog box opens.

  1. Verify that the metadata that Bloom supplied is correct.

    • If any is not correct, edit your book, change Copyright and License selections and so on.

  2. Type metadata in each text box field that Bloom does not supply.

  3. Click the Subjects down arrow ().

Do these steps in the drop-down subject list that appears:

    • Scroll and expand () subjects to navigate to the desired one.

    • Select () one or more subjects.

    • Click anywhere outside of the list to close the drop-down list.

To remove a subject you had selected, click the small gray x () at the end of the subject or display the list again and clear () its check box.
