Save ePUB file

Review the information in Publish as ePUB.

Then, do these steps:

  1. Click the Publish tab, and then select () ePUB.

  2. Click the ePUB mode control and choose a different mode, if you are permitted.

  3. If you chose the Flowable mode, you can select () or clear () Include image descriptions on page.

  4. Click Accessibility Checker and then check your book in the Bloom Accessibility Checker dialog box.

  5. Click Bloom Metadata and then add metadata about your book in the Book Metadata dialog box.

  6. Click SAVE.

  7. In the Save As dialog box, navigate to the folder where you want to save the file.

    • Edit the file name (optional). Do not change type of file. It needs to be *.epub.

    • Click Save.

    • If you saved the ePUB file in the book folder, use the command Show in File Explorer to find it.

Tip

  • Click  Refresh at any time to update the preview.

  • Below Help, you can click a link to open a topic in this Bloom Help system.

  • Dropbox and similar services are a handy way to move your Bloom ePUB files to your phone.

Save the ePUB file to your Dropbox folder. On your phone, open the Dropbox app and select your book.
