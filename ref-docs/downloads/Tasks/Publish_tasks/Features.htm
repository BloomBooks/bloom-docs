About Book Features

In BloomLibrary.org, Book Features displays a set of features and how many books have each of those features. In the library, your publications are included in these groups, based on Features options selected in the Publish tab.

 Note: For BloomPub, only Motion Book directly affects your publication.

The other features listed there match what you in see in Web so you do not need to make these selections multiple times.

Web and  BloomPUB

  •  Talking Book is automatically selected () if you select () one of the Talking Book Languages.

  •  Sign Language is automatically selected () if you have set a sign language and the book has at least one video.

If this Sign Language check box is not available, a tool tip appears with a brief explanation. You can click the underlined text to help you enable this feature. 

You can clear () that check box if you do not want to declare this book as a sign language book. For example, the video might not be sign language. 

  •  Activity is automatically selected () if the book has activity (Game Pages) pages.

  •  Comics is automatically selected () if the book has overlays.

You can clear () that check box if you do not want to declare this book to be a comic.

Books that are not comics can have overlays.

  •  Motion Book - You can select () this if the book has motion.

In BloomPUB, orientation buttons appear next to the preview.

If you click the landscape option, then you will see the motion in the preview.

Your section here affects whether the reader can change the orientation to landscape and see the motion. 

  •  Accessible to the Visually Impaired - You can select () this if the book has image descriptions.

You might refer to information related to the Book Metadata dialog box or accessibility if you are not sure whether you should select this feature.

Audio or Video

For Audio or Video, select () Motion Book to regenerate the preview and then record the book with the motion.  

ePUB

Use the Book Metadata dialog box to indicate whether your book has accessibility features Has Image Descriptions or Sign Language.
