Create a shell book for others to use

Do the steps below to make a shell book that other people can use to make books in their collections.

  1. Open the collection that uses the language or languages you want the shell book to have.

  2. In the Collections tab, click the appropriate source.

You can click Arithmetic, Basic Book, Big Book or another template or specialized template that was installed with Bloom.

  1. Click MAKE A BOOK USING THIS SOURCE.

The new book opens in the Edit tab.

  1. Tasks overview has links to Collections tab tasks and Edit tab tasks, which list steps you can reference.

When you are done, minimally, you should have:

    • typed in the appropriate metadata

    • added all the intended pages

    • added an Translation Instructions page with instructions, if you need that page

    • typed in all the source text in one or more language

    • configured  all the intended styles

    • chosen all the pictures that you want to appear in the published book.

  1. Make the shell book available to other people in any of these ways:

    • Publish the shell to BloomLibrary.

    • Make a Bloom Pack.

    • Make a Reader Template Bloom Pack.

This allows others to get the shell and make a book from it.
