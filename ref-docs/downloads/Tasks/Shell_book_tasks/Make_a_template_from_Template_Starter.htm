Make a template from Template Starter template

  • Review the information in Template Starter.

The specific content that you include in your template can vary a great deal, so consider some of these steps as possible things you can do. For example, you may or may not want standardized text.

  1. Click Collections tab.

  2. Use any of the tabs in the Settings dialog box to specify languages, fonts and other settings.

  3. Below Sources For New Books, click the template called Template Starter.

A page from the template starter and important How to use the Template Starter information appear in the main pane. Review this information carefully.

  1. Click MAKE A BOOK USING THIS SOURCE.

The book is added and it opens in the Edit tab.

  1. On the toolbar, do one or both of these steps:

    • Select () languages.

    • Choose a page size and orientation.

  2. On the Image For Thumbnail: TemplateStarter Front/Back Matter page, choose a picture.

  3. On the About page, do these steps:

    • Type a title for the template in the Template Title box in each language for which there is a box.

    • Type your name in the Name of Template Creator box.

  4. Add a page.

Then, do any of these steps:

    • In the upper-left corner of the page, type a useful page label that describes the layout of the particular page. You may want to look in the Add Page dialog box to see how other pages are labeled in Bloom.

    • Using the Format dialog box to set text box formatting.

    • Use the Change Layout control to open the Text Box Properties dialog box.  For example, you might want to add hint bubbles.

    • You might want to duplicate the page after the layout is set.

    • Type words in the text boxes that you want to include in all books made from this template.

    • Insert a picture that you want to include in all books made from this template.

    • Repeat this step for each page.

  1. Optionally, add a description that will appear in the right pane of the Collections tab when someone clicks your template.

    To do this, put a text file named ReadMe-en.md in the folder for your template (where en is English).
    This file should follow the markdown standard and can include screenshots. To find your folder, see Right-click menu commands - Collections tab.

Note

  • For a description in another language, make a translated version of that file and change the -en to another language's two letter code (e.g., ReadMe-fr.md for French). It will appear when the UI is set to use that language (e.g. French).

  • If you want Add Page to also provide a short description of the page, close Bloom. Then, edit the html for the template in Notepad. You might want to get more help.

  • The cover pages, title page and credits page are added when you use your template it as a source for a new book.
