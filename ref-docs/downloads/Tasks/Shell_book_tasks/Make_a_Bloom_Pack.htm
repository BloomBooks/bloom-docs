Make a Bloom Pack

Make a Bloom Pack of Shell Books

Do these steps to make a Bloom Pack that contains all the shell books in the collection that is open:

  1. Click the Collections tab, if it is not already the current tab.

  2. In the left pane, click the down arrow () that appears with the collection name and then click Make a Bloom Pack of Shell Books.

The Save As dialog box appears.

  1. Click the folder where you want to save the file.

If possible, click a folder on a USB drive or similar to make it easier to share with others.

  1. Edit the file name, if necessary, but do not change the Save as Type selection.

  2. Click Save.

A compressed file (.BloomPack) is made that contains all the shell books and templates.

Save a book as a Bloom Pack

Do these steps to save a book as a Bloom Pack:

  1. In the Collections tab, click the book you want to save as Bloom Pack.

  2. Click the down arrow () that appears below the book thumbnail, point to More and then click Save as Bloom Pack (*.BloomPack).

  3. In the Save As dialog box, click the folder where you want to save the file.

If possible, click a folder on a USB drive or similar to make it easier to share with others.

  1. Edit the file name, if necessary, but do not change the Save as Type selection.

  2. Click Save.

A compressed file (.BloomPack) is made that contains the book.

Note

  • Bloom Pack discusses ways to distribute the file.

  • See Also: Make a Reader Template Bloom Pack.
