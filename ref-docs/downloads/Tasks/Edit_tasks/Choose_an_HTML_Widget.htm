Choose an HTML Widget

You can add a Widget Page. Or, in other pages, you can use the Change Layout controls to add a box that can hold an HTML Widget.

Then, do these steps:

  1. On a page with a widget icon (), click (Choose Widget).

  2. Find and click the widget file you want (*.wdgt or *.html file).

  3. Click Open.

  4. The widget appears.

How to use your widget is beyond the scope of these Bloom Helps.

https://docs.bloomlibrary.org/html5-widgets/ provides more information about widgets.
