Copy and paste a page

Duplicate Page copies and immediately pastes a page into the same book.

Alternatively, you can copy a page to your computer's clipboard. Then, open another book in the same collection and paste the page into that book.

Do these steps after you are done formatting, using Change Layout control and working with styles.

  1. Click Collections tab.

  2. In the left pane, click the book that has the page you want to copy. Then, do one of these steps:

    • Double-click the book.

    • Click the Edit tab.

    • Click EDIT THIS BOOK.

The book opens in the Edit tab.

  1. In the Pages (left) pane, click the page you want to copy.

  2. Right-click that same page or click the down arrow, and then click Copy Page.

  3. Click Collections tab again.

  4. In the left pane, click the book into which you want to paste the page. Then do one of these steps:

    • Double-click it.

    • Click the Edit tab.

    • Click EDIT THIS BOOK.

The book opens in the Edit tab.

  1. In the Pages (left) pane, click the page that you want to come before the page you will paste.

  2. Right-click that same page or click the down arrow, and then click Paste Page.

A copy of the page you clicked appears in the Pages pane.

If needed, you can reorder pages.

Important

  • If the page you copied and pasted had an image, Bloom renames the pasted image if the book already had one with the same name.

  • Any styles you added or changed are copied into the other book.

  • See Also: Template Starter.
