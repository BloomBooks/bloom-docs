Set up games overview

Game Tool overview describes controls that appear in the Games Tool tool box. 

Here are some controls that you might see on the game page you added. Use to Set up Drag Images to Targets game as a model of how you might set up your games. 

When you click an item on a game page, a toolbar appears with a set of options, such as
  or   or .

The more button () opens a list with these and also more options that you can use. These are described below.

 and  

  • Click  Format Text to open and use the Format dialog box.

  • Click  Copy Text to copy selected text to your clipboard.

  • Click  Paste Text if you already copied text on to your computer clipboard.

  • Select ()  Auto Height to let Bloom adjust the height of the text box automatically. Click it again to remove the check mark if you want to manually set the height.

  • Select () Part of the right answer to set this box as the right answer. Click it again if it is not part of the right answer and remove the target box.

  • Click  to open the Talking Book Tool so you can record the words.

  • Click  Duplicate to make a copy of the text box.

  • Click  Delete to delete the current text and the text box.

and 

  • Click  Choose image from your computer and then choose the desired image.

  • Click  Paste image if you already copied an image on to your computer clipboard.

  • Click  Copy image to copy the current image to your clipboard.

  • Click  Set image information to open and use the Copyright and License dialog box. If the image is locked, you can only view the image credits, Copyright and License information.

  • Click  Reset image to reset the size after you cropped the image to be smaller.

  • Click Paste Hyperlink  or paste a hyperlink from your clipboard. The url will open when when this book has been published, You can preview the link in the Publish tab.

  • Select () Part of the right answer to set this image as part of the right answer. Click it again if it is not part of the right answer and remove the target box.

  • Click  and then choose the sound file that you want to play when the item is touched. For example, if you touch or click the image in the Bloom Player. Otherwise, click None.

  • Click  Duplicate to make a copy of the image and the image box.

  • Click  Delete to delete the current image and its box.

  • Click  Choose Video from your Computer and then choose the desired video (usually a sign language video).

  • Click  Record yourself to open and use the  Sign Language Tool.

  • Click  Play Earlier or  Play Later to move the focus to another video if you have two or more video.

  • Select () Part of the right answer to set this video as part of the right answer. Click it again if it is not part of the right answer and remove the target box.

  • Click  Duplicate to make a copy of the video.

  • Click  Delete to delete the current video and the video box.
