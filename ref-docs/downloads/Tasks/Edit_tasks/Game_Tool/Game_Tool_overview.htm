Game Tool overview

The Game Tool is one of the tools in the tool box. Some of the game pages you can add make the Game Tool appear, but not all of them. In addition, some of the game pages do not use all of the features the Game Tool. This is a Bloom Enterprise feature.

When this tool is shown, and if the current game page allows it, you can see various items you can drag onto a game page. Objects to Drag are items that reader can click and drag to a target. Objects that do not Move are fixed in place so the reader cannot move them. Set up games overview describes the various controls you see on game pages.

Here are things you can see in the Game Tool tab:

This adds an image placeholder so you can add an image to the page. There is one that can be dragged by the reader, and one that is fixed in place.

This adds a video placeholder so you can add a sign language video to the page. There is one that can be dragged by the reader, and one that is fixed in place.

This adds a placeholder that can hold words and an audio file that will play a sound with the item is touched.

This adds a placeholder that can hold a *.gif image, which can be animated.

This adds a placeholder that can hold an audio file that will play a sound with the item is touched.

This add a square to the page that you can fill or leave empty. You can add it as a border around other items.

  • Theme allows you to choose the color theme for the objects and background.

  • Options has these three buttons:

  •  Keep all items and targets the same size. In this case, if you resize an image or a target, all of them change size. 

 in this case, you can resize an image and its target without changing the size of other images or targets.

  •  Show drag targets during Play. In this case, the people can see the targets that they will drag letters or images into when they play the game.

 In this case, the people will not see the target and will just drag and drop the letters or images in a general area when they play the game. 

  •  Show Answer even during Play. In this case, the target shows the correct answer. When a person drags a letter or image into the target box, the box disappears. 

 In this case, the targets are empty until a person drags a letter or image into the target box. 

Above some game pages, you can see these controls:

  • Game Setup Mode: Start, Correct, Wrong and Play. You can click these modes to see how the page will appear when the reader is playing the game and the results the reader will see.
