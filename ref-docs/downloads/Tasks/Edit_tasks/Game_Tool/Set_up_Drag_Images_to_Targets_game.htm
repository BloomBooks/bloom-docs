Set up the Drag Images to Targets game

Note:

  • There is not a Help topic for each possible game page you can add. Instead, use this topic as a model. There will be differences but the process is generally similar.

  • Review Game Tool overview and Set up games overview.

Do any of these steps:

  1. In the Pages pane, click an inside page that is a Drag Images to Targets page.

For this game page, the goal is to allow a reader to click and drag images into targets. Images could be numbers, letters or other things.

  1. Click Start in the Game Setup Mode control, if it is not already selected.

  2. In the Game Tool tab, click the item you want to add to the game page and drag it to the place on the page where you want it to appear.

  3. Click  to add a background image if desired.

  4. Do any of the options described in Set up games overview, including the theme and other options.

  • Click and drag the end   or a corner of an item to resize it, or click and drag the entire item to move it on the page.

  1. Click Correct in the Game Setup mode. Use any of the controls to change what the reader sees and hears when they get the correct answer.

  2. Click Wrong in the Game Setup mode. Use any of the controls to change what the reader sees and hears when their answer is not correct.

  3. Click Play in the Game Setup Mode control, and then play the game.

  • Click  to check the results from playing the game.

  • Click  to play the game again.

  • Chick  to see the correct answers.

Then click Start again to continue. Use any of the controls to change the game until it is the game you want readers to experience.
