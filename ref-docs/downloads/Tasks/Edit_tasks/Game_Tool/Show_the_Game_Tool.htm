Show the Game Tool

Not all of the game pages you can add will cause the Game Tool appear.

  1. Open the book in the Edit tab.

  2. Click a game page that uses the Game Tool.

  • If no existing page uses this tool, add one.

  1. Look at the right side of the Bloom window and then do these steps:

    • If you see the button, click it to expand (open) the tool box.

    • Click the Game Tool tab, if is available but not shown already.

The Game Tool controls appear.

Now you can set up the game on the current page.

Note

  • Some game pages you can add do not use the Game Tool.
