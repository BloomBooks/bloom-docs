Motion Tool overview

The Motion Tool is one of the tools in the tool box.

More specifically, this tool allows you to set a basic motion to an otherwise still picture. The motion is a pan-and-zoom from rectangle 1 to rectangle 2. You see these rectangles when you use the Motion Tool. You can add motion to the first picture on each inside page.

Tip

  • You can publish your motion book to the Bloom Reader App.

  • It is best that you record the audio before you use this tool. This will help the durations match and the Preview will include the highlighted text.

Similarly, you can also include the background music when you preview the Motion results.

Using the Preview feature can also help you determine the relative volume levels of the recorded audio and background music.
