Sample Words tab

In the Set up Decodable Reader Tool dialog box, the Sample Words tab has two controls and two boxes.

  • We are using lists of allowed words to define stages

If you select () this control, the two boxes described below become unavailable.

See: Allowed words files.

  • We are using letters with sight words to define stages

If you select () this control, you use these two boxes:

1) Type Words Here

In this box, type words that you want Bloom to suggest. These words are in addition to sight words and to words in the files.

 

2) Place Text Files in Your Sample Texts Folder

You cannot type in this box. This box shows names of files. These are files that were saved in the Sample Texts folder.

Click the Sample Texts Folder link to open the folder. Use the file explorer to copy sample texts into the folder. These files must be in UTF-8 format and saved as *.txt files.

Note

  • See Also: Make Bloom suggest words for a decodable reader.

  • https://vimeo.com/channels/bloomlibrary/121688809 is a video that can help you learn about sample words.

  • You can set the minimum font size for letters and words that appear in the tool box and its related dialog boxes. In the Boom Making tab of the Settings dialog box, click Special Script Settings for the language and then use the Font size when displayed in tools control.
