Using the Music Tool

  1. In the Pages pane, click the first page.

  2. Click the Music Tool tab in the tool box.

  3. In the Music Tool tab, select () No Music or Start new music.

  1. To start new music, do these steps:

    • Click the Choose link ().

    • In the Open dialog box, click the music file that you want and then click Open.

The file name appears after the play button ().

    • Click the play button to hear the music and any recorded audio on this page.

The play button () changes to become a pause button ().

    • Adjust the volume with the slider ().

This volume slider changes the volume of the music, but not the recorded audio. The numerical value is logarithmically related to the visual slider so that you have finer control at lower volumes.

    • To stop the playback, click the stop button.

  1. In the Pages pane, click the next page.

If a previous page played music, the default selection is Continue music from previous page.

If you want the music to continue, then click a subsequent page for which you want to have either no music or new music.

    • In the Music Tool tab, select () one of these options:

      • No Music

      • Start new music.

Repeat the steps above to start new music.

  1. Repeat these steps until each page either plays the desired music or is set to play no music at all.
