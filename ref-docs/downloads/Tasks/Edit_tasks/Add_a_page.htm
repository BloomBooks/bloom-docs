Add a page

The Add Page () button allows you to add an inside page.

  1. In the Pages (left) pane, click the page that you want to precede the new page or pages.

The page you clicked is now the current page and has a yellow border.

  1. At the bottom of the Pages pane, click the Add Page button.

The Add Page dialog box opens.

  1. On the left side of the dialog box, click the page that has the layout that you want.

The page is described on the right side of the dialog box.

  1. Edit the numerical value (Number of pages to add), which is next to the ADD PAGE button.

  2. Click the ADD PAGE button.

The new page or pages are added and appear after the page you previously selected.

  1. If you added the Custom (blank) page, add items to it.

Important

  • (dot) — You need a Bloom Enterprise subscription to add the page.

  • Keyboard shortcut: Ctrl+N adds a new page without displaying the Add Page dialog box.

If you already used the Add Page dialog box to choose a page, Bloom will remember that page and add it again. Otherwise, Bloom adds a Basic Text & Picture page.

  • Some users may want to use the Translation Instructions page.
