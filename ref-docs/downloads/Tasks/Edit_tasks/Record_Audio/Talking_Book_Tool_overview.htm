Talking Book Tool overview

The Talking Book Tool is one of the tools in the tool box.

You can make an audio recording of a book translated into the Local Language.

You can record image descriptions and Overlay Tool bubbles, text blocks and captions.

Recordings can be heard on ePUB readers and with the Bloom Reader App.

If aeneas / eSpeak are installed, you can record an entire text box and let Bloom adjust timings in your recordings by sentence for highlighting. Otherwise, you can only record one sentence at a time.
