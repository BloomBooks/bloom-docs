Using the Talking Book Tool

  1. In the Pages pane, click a page that has words in a text box.

This includes text boxes, captions and speech bubbles added to images and can include image descriptions. You may want to start with the text box for the book title on the Front Cover page.

  1. Click  Advanced to see the options. See: Advanced controls. 

  2. Press and hold 3) Speak and then read the highlighted text. When you are done with that text, release the button.

  3. Click 4) Check to listen to that recording.

    • Pause () the playback if you want to.

    • If you are not satisfied with the recording, click and hold the Speak button again and read that text again.

Clear is an optional way to delete the recording for the current text.

    • If you are satisfied with the recording, continue to the next step.

  1. Click 5) Adjust Timings if it is displayed. See: Adjust Timings.

  2. Click Next ( or ) to move to the next text, if any.

    • To return to previous text, click Back until the desired text is highlighted. Then you can check it again and record it again if needed.

  3. Click Listen to the whole page to make sure all recordings are acceptable.

Tip

  • The Play button in the Music Tool, allows you to hear the relative volumes of the recorded audio and background music.
