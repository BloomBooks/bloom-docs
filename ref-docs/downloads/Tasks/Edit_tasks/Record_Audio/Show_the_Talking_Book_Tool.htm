Show the Talking Book Tool

  1. Open the book in the Edit tab.

  1. Look at the right side of the Bloom window.

    • If you see the button, click it to expand (open) the tool box.

    • Click the Talking Book Tool tab.

The Talking Book Tool controls appear.

Now you can use this tool.
