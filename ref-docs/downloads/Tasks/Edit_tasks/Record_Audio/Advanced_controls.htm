Advanced controls

You might want to review Adjust Timings and Fine-tuning Aeneas Timings on the Internet before you continue. 

  1. In the Pages pane, click a page that has words in a text box.

This includes text boxes, captions and speech bubbles added to images and can include image descriptions. You may want to start with the text box for the book title on the Front Cover page.

  1. Click  Advanced.

Consider these options.

  • The  Insert Segment Marker button is displayed so where you need to manually insert a segment marker, you can click it.

  • Recording Mode: You can select () By Sentence, and then insert segment markers manually. Alternatively, if you want to record all the sentences in a text box at the same time, select () By Whole Text Box. Then you can adjust timings.

 Import Recording is enabled when By Whole Text Box is selected. Click it to find and import an existing audio file for the current text box.

  • Enable () Show playback order buttons if you need to set the playback order. 

Then, disable () that feature again to make the other Talking Book Tool controls available again.

  • Select () Show image description if you want to record the image descriptions.

Tip

  • Advanced options can selected differently for the different pages in your book.

  • You cannot change the Recording Mode selection, after you have recorded text.

If you need to change it, click Clear to delete the current audio recording. 

  • The Play button in the Music Tool, allows you to hear the relative volumes of the recorded audio and background music.
