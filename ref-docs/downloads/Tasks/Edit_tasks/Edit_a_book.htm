Edit a book

Do any of these steps when you edit a book or shell book.

  1. Click the Collections tab.

  2. In the left pane, click the book you will edit.

    • If the book is in a Team Collection, the book must be available for you to check out ().

  3. Click PREVIEW and then click EDIT THIS BOOK.

It opens in the Edit tab. You will see some Bloom callout boxes.

  1. On the toolbar, change the page layout or select languages.

  2. In the Pages pane, click the page you will to edit. Do any of these steps:

Front Cover pages:

  • Edit the book title, change the picture or choose a topic.

  • Type or edit the words in the Inside Front Cover page.

Title Page:

  • Edit the book title, contributions or acknowledgements.

  • Click the Paste Image Credits link to update the image information. You may need to select and delete credit information that was previously pasted there, but not information that you manually typed into that text box.

Credits Page:

  • Use the Copyright and License dialog box.

  • Type the ISBN, if you have one.

  • Edit the acknowledgements.

Inside pages:

  • Edit existing words or type new words.

  • Add a page.

  • Change the page layout (if permitted).

  • Change a picture.

  • Change picture metadata.

  • Remove a page.

  • Reorder pages.

  • Use any of the tools in the tool box.

Back Cover pages:

  • Type or edit words that you want to appear there.

  1. If you are working in a Team Collection, check in the book (). Then, other team members can see your changes.

    • In some cases, you might want to click and then click Forget Changes & Check in Book.

Tip

  • Formatting Text overview help you work with styles and formatting.

  • A red font color or red line means that the text box is too full.
