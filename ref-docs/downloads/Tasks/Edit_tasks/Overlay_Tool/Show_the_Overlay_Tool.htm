Show the Overlay Tool

  1. Open the book in the Edit tab.

  2. Look at the right side of the Bloom window and then do these steps:

    • If you see the button, click it to expand (open) the tool box.

    • If you do not see the Overlay Tool tab, click the More tab.
      Then select () Overlay Tool.

    • Click the Overlay Tool tab.

The Overlay Tool controls appear.

If you see the BLOOM ENTERPRISE SETTINGS button, click it so you can enter your Subscription Code to enable the Overlay Tool.

Now you can use this tool.
