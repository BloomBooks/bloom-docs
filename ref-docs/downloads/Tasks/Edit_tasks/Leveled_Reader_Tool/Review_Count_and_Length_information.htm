Review Count and Length information

In the Leveled Reader Tool tab, you see Word Counts, Word Lengths, and Sentence Counts.

About Count and Length criteria describes how you select the count and length criteria, and set Max values when you set up levels.

Here is what you do with the resulting information:

  • Review the count and length numbers and their colors for the current level.

Numbers in the Actual columns can change color:

    • Green: Actual is less than or equal to Max.

    • Orange: Actual is greater than Max. You need to make word or sentence changes to correct them.

Note

  • An orange background also appears in sentences that are too long (example). You need to edit sentences to reduce the number of words.

  • An orange border appears around the page when there are too many words on the page.

  • You can click COPY BOOK STATS to copy them to your clipboard.

  • https://vimeo.com/123325405 is a video that can help you learn about the Leveled Reader Tool.
