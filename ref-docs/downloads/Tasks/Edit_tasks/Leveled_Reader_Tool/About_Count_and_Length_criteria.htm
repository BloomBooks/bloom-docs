About Count and Length criteria

You review Word Counts, Word Lengths and Sentence Counts in the Leveled Reader Tool. These include Max (Maximum) numbers and Actual numbers for the current level. Some are for the current page and others for the current book. Some are averages.

When you configure a level in the Reader Levels tab, you choose what you want to see there.

The Reader Levels tab has a table and a list of possible criteria:

  • Each row is a level.

  • Each column corresponds with one criterion in the list.

  • Each cell (the intersection of a row and column) contains a number or a dash (-).

For each criterion you select () in the list, you must type a number. The number is the maximum count or length for that criteria. It appears in the corresponding cell.

For each criterion you clear () in the list, the cell shows as dash.
