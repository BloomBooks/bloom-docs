Punctuation tab

For Bloom to accurately determine the word count information, it must know all the sentence-ending punctuation. Bloom already knows which characters in the world's languages always mark the end of sentences, but you can add more.

  • Type the Unicode character escape sequences you need to add, separated by a space.

For example, \u0020 tells Bloom that normal spaces break your sentences.
