Show the Leveled Reader Tool

This tool is selected and displayed by default in books made from the Leveled Reader template. If you do not see this tool in your book, do these steps:

  1. Open the book in the Edit tab.

  2. Look at the right side of the Bloom window and then do these steps:

    • If you see the button, click it to expand (open) the tool box.

    • If you do not see the Leveled Reader Tool tab, click the More tab.
      Then select () Leveled Reader Tool.

    • Click the Leveled Reader Tool tab.

The Leveled Reader Tool controls appear.

Now you can use this tool.
