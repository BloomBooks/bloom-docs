Set up levels

  1. In the Leveled Reader Tool tab, click Set up Levels ().

The Set up Leveled Reader Tool dialog box opens.

  1. Use the Reader Levels tab to add, configure, reorder or remove levels. This includes For this Level statements.

  1. Use the Punctuation tab if you need to add more special sentence-ending characters. This helps Bloom determine the word count information.
