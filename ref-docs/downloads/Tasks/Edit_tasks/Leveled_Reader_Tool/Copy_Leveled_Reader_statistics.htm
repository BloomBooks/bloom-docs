Copy Leveled Reader statistics

  • In the Leveled Reader Tool tab, click COPY BOOK STATS.

The book title, level and Actual column statistics are copied to your computer's clipboard.

They are tab-delimited so you can paste them in a spreadsheet. There will be a header line and a data line.
