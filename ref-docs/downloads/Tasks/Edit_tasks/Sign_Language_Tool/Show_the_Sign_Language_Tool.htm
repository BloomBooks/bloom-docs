Show the Sign Language Tool

This tool is selected and displayed by default in books made from the Sign Language template. If you do not see this tool in your book, do these steps:

  1. Open the book in the Edit tab.

  2. Look at the right side of the Bloom window and then do these steps:

    • If you see the button, click it to expand (open) the tool box.

    • If you do not see the Sign Language Tool tab, click the More tab. Then select () Sign Language Tool.

    • Click the Sign Language Tool tab.

The Sign Language Tool controls appear.

Now you can use this tool on inside pages that can hold videos.

Tip

  • If you click a page that has an existing video or a video placeholder, you can click the video or placeholder. Then the Sign Language Tool appears in the tool box.
