Using the Sign Language Tool

On pages that can hold a video, you can import an existing video or do the steps below to record a new video:

  1. Review Video recording tips.

  2. In the Pages pane, click a page.

  3. Click the Sign Language Tool tab in the tool box if it is not already shown.

  4. In the Sign Language Tool tab, do these steps to record a video:

    • If the page can hold more than one video, click the video placeholder that is in the location on the page where you want to have the video.

    • Click (Start Recording).
      The three-second countdown begins.
      A colored border appears around the small view window while Bloom is recording. A timer shows you how long you have been recording.

  5. When you are done recording, press any key on your keyboard or click .

  6. Play the video ().

  7. Trim the video if desired.

  8. Use any of the Advanced features:

    • Import

    • Show In Folder

    • Delete.

Click Advanced to hide them.
