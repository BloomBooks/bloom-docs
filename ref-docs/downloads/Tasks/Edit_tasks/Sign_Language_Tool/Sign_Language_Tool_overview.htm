Sign Language Tool overview

The Sign Language Tool is one of the tools in the tool box.

It is usually used with the Sign Language template. But you can use it to work with any page that can hold a video.

Video recording tips discuss ways to help you make high-quality videos. Near the top of the tool a small view windows shows you what your camera sees.
