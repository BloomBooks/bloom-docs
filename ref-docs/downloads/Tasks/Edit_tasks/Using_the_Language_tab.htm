Using the Language tab

Open the Text Box Properties dialog box.

Then, do these steps to choose the language for this text box:

  1. Click the Language tab.

  2. Select () the language option you want to use for the current text box.

    • To make the text box use only one language, select one of the language options.

    • Select Normal to clear the language option. Then it will use languages as set in the Languages tab of the Settings dialog box.

  4. Click outside of the dialog box to close it.

  5. Click the end of the Change Layout control to hide the controls.

Important

  • Unlike setting a style, this property is for the current text box only. It will persist if you use Duplicate Page. However, if you change it later in one of the pages, it will not change in any other pages.

  • In the Settings dialog box:

    • Use the Book Making tab to work with font and script settings for languages. You can also choose the language you want for page numbers in that dialog box.

    • Use the Languages tab to change or set languages for the collection.
