Template

A template is anything that serves as a pattern.

  • In Bloom, a template is a set of pages you can use when you make a new book.

In the Collections tab, below Sources for New Books you see:

  • Templates: Basic Book and eBook.

  • Specialized Templates: Arithmetic Template, Big Book, Decodable Reader, Digital Comic Book, Leveled Reader, Paper Comic Book, Sign Language, Template Starter and Wall Calendar.

  • Page templates are available in Add Page dialog box when you add pages.
