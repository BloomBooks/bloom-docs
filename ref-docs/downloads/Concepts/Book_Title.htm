Book title

A text box that holds the book title appears on both the Front Cover page and on the Title Page.

  • You can type, edit or format () the title of the book on those pages.

  • Language 3 is not used for titles.

  • The book title appears in the Book Metadata dialog box, but not in the Language 2 or Language 3.
