Video / Video Placeholder

A video placeholder () means that the page that can hold a video.

Sign Language describes that you can add pages with video placeholders.

Tip

  • Click a video placeholder () if you do not see the Sign Language Tool in the tool box.

  • The Sign Language Tool displays metadata about the current video. You can record a video in Bloom or import them.

  • You can use Change Layout controls to insert a video placeholder.

  • You should not have more than one video per page.
