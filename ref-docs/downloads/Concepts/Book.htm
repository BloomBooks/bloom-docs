Book

In these Bloom Helps, the generic term "book" means any document in a collection.

This can include any of the following:

  • Any book made from a template.

  • Any book made from a sample shell or downloaded book.

  • A Wall Calendar or Template Starter.

Typically a book is made up of pages.
