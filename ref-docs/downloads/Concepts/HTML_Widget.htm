HTML Widget

An HTML Widget is a small computer program that runs in a browser. A widget can be available as an *.wdgt or an *.html file.

https://docs.bloomlibrary.org/html5-widgets/ provides more information about widgets.

If you are set up with a Bloom Enterprise subscription, here are things you can do:

  • You can add a Widget Page and then choose one.

  • For other existing pages, use the Bloom Change Layout controls to add a box that can hold an HTML Widget. Then, choose one for that box.
