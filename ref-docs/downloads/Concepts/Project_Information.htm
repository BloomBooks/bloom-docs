Project Information

The Project Information tab is in the Settings dialog box.

Project information can include any of these data:

  • Country, province and district names

  • the Bloom collection name

Tip

  • Enter this data for each collection you have. It is not passed between computers in Bloom Packs.

  • Project information can appear on the title pages, credits pages and back outside cover.

  • Your Bloom Subscription Code is also project information.
