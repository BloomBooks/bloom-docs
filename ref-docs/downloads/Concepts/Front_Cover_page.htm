Front Cover page

In the Edit tab you see the Pages pane. The Front Cover and Inside Front Cover pages are shown there. The Front Cover page is the top outside page when a printed book is closed and you look at it from the front. For a PDFs or ePUBs, it is the first page.

The Front Cover contains:

  • Book title - it can appear in the top two languages.

  • Picture - it appears on the front cover.

  • A text box so you can show the author/illustrator and so on. The Title Page has a text box for contributions.

  • Languages used in the book. If the book contains a video, then also the name of the sign language.

  • Topic of the book.

The Inside Front Cover can contain:

  • Any additional information that you want the reader to see.

You can also use the back cover pages.

Note

  • The book title and picture appear in the Book Metadata dialog box. The book title is metadata when you Publish to Web.

  • You can format () the text boxes on these pages.

  • Big Book and Wall Calendar are slightly different.
