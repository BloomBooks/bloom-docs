Metadata in Bloom

Metadata is "data about data." That is, it describes the data, which in this context is a book. Metadata helps people find and learn about books, and about license information. Metadata contributes to accessibility.

In Bloom, metadata includes:

  • Acknowledgements

  • Content collected or entered in the Book Metadata dialog box

  • Contributions

  • Copyright and license -Images

  • Copyright and license - Text

  • Enter your Subscription Code

  • ISBN

  • Project Information

  • Publish to Web

  • Title

  • Topic
