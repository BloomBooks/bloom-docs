Flagged words in a reader

In a decodable or leveled reader, words can be flagged with a colored background. Also, an entire page can get an orange border. A yellow pop-up box appears to explain why the word or sentence is flagged.

Examples

  • This word is not decodable in this stage.

This word uses letters that have not yet been introduced.

(Be aware that a word that has too many letters can look similar.)

  • This sentence is too long for this level.

The number of words is more than the maximum number set for the current level.

  • Actual per page is larger than Max per page.

An orange border means that there are too many words or unique words on a page for the current level.
