Bloom Pack

There are two kinds of Bloom Packs.

Bloom Pack (basic)

  • Make Bloom Pack of Shell Books

When you make one, here is what happens:

    • Bloom puts all the shell books in the collection into a zip file.

    • Bloom changes the file name from <name>.zip to <name>.BloomPack.

One person can create shell books for others and then make a Bloom Pack that other people can get. People who get these Bloom Packs can translate the book into other languages. However, the Template Pages page is not available for use, nor can the pages be reordered.

  • Save as Bloom Pack (.BloomPack)

This command is available in any collection.

Use it to save one book in a .BloomPack file.

Reader Template Bloom Pack

  • Make Reader Template Bloom Pack

Use this command when you work with the Decodable Reader Tool or Leveled Reader Tool.

When you make one, here is what happens:

    • Bloom does the same things as it does for a basic Bloom Pack but also puts the decodable reader settings, the leveled reader settings, and the current set of sample words in the file.

    • Files in the Sample Texts folder become .json files. See: About word list and sample texts.

One person can set up decodable readers or set up leveled readers and then make a reader template Bloom Pack. Other people can get it. In this case, the people who get these Bloom Packs can use them like templates. You can add pages and reorder pages.

Distribute a Bloom Pack file in any of these ways:

  • Put it on a shared USB flash drive.

  • Put it in a shared folder on a networked computer.

  • Use Dropbox (https://www.dropbox.com/).

  • Send it with Skype (or similar), if it is not too big.

  • Attach it to an e-mail, if it is not too big.

Delete a Bloom Pack folder from your computer

  • If you installed a Bloom Pack, the Collections tab can display a folder button () near the name of the collection.

Click the button to find the collection folder you want to delete.

Note

  • Bloom Pack files typically have this icon: .

  • Bloom Subscription Codes are not included in Bloom Packs. So recipients of your Bloom Pack need to do the steps above before or after using the contents of your Bloom Pack.

  • On the (Get Help) menu, click Building Reader Templates and Using Reader Templates. These documents describe these templates and how to use them.
