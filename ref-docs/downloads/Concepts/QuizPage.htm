Quiz Page

In the Add Page dialog box, you see Quiz Page below Game Pages.

Quiz Page is an interactive page. You can add one or more and use them to ask comprehension questions and provide possible answers.

Instructions for each text box appear in a yellow box.

Example

When this page is used with Bloom Reader, your project's dashboard can show aggregated comprehension scores.

Quiz pages are not included in ePUB publications.

Tip

  • The dot () means this is a Bloom Enterprise feature.

  • You can reorder pages in the Pages pane.
