Title Page

In the Edit tab you see the click Pages pane. The Title Page is shown there.

The Title Page contains:

  • Book title

  • Contributions

  • Acknowledgements

  • Languages used in the book. If the book contains a video, then also the name of the sign language.

  • Project Information and any branding.

Note

  • You can format () the text boxes on this page.

  • If you select the CC0 Public Domain license, the Copyright Year and Copyright Holder are hidden if you click Paste Image Credits.

  • This page cannot use Language 3.

  • Books made with the Digital Comic Book Template have a different page order.
