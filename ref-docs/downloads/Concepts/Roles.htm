Roles

Bloom supports roles. This means that people can have different tasks to do. Of course, each situation is different and anyone can author new books without any established set of roles. https://vimeo.com/album/3146448/video/114043219 is a video that can help you understand possible roles.

Here are some possible roles and tasks. You might work in a group with fewer or more roles, or with different task assignments.

Shell book and template provider

  • Create shell books that store words in one or more language.

  • Set up styles and text box properties.

  • Download shells from or upload shells to Bloom Library.

  • Create templates.

  • Distribute shell books and templates as Bloom Packs.

Advisor

  • Install and set up Bloom on a computer that will be used by a translator.

  • Create a Bloom collection.

  • Set up Decodable Reader

  • Set up Leveled Reader

  • Protect the settings.

  • Change the UI language.

  • Get ISBN numbers for books, if they are necessary.

  • Get and install Bloom Packs.

  • Download books from Bloom Library (more)

  • Get and install Front/Back Matter Packs.

  • Setup the keyboards. Teach the translator how to use them.

  • Publish the translated books.

Translator (speaker of the local language)

  • Make books from shell books.

  • Read the source text. Then, type the translation of it in text boxes that use the local language.

Administrator of a computer lab

  • Install Bloom for all users of a computer.
