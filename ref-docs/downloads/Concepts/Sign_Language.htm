Sign Language

A sign language is a system of communication using visual gestures and signs.

A signer is someone who communications information to deaf people using a sign language.

Here are items you can use in Bloom to include videos of a signer in your book.

Sign Language - Setting dialog box

  • Use the Languages tab to set the sign language you will use in videos.

See: About Book Features.

Sign Language - Template

  • This is one of the templates you can use as the basis of a book. When you click this template, information about the template appear in the main (right) pane.

When you open it in the Edit tab, the Sign Language Tool opens in the tool box.

Sign Language - Add Pages

  • In the Add Page dialog box you see various pages with a video placeholder below the Sign Language heading.

You can import or record videos in Bloom when you use these pages.

The dot () means that a Bloom Enterprise subscription is required.

Sign Language Tool

  • This tool provides the controls you need to work with videos.

Important

  • You should not have more than one video per page.
