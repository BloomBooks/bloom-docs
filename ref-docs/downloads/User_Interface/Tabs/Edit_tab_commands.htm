Edit tab commands

Pages (left) pane

The Pages pane down arrow example shows the down arrow you can use.

  • Duplicate Page - This command copies and immediately pastes (duplicates) a page in the same book (more).

  • Duplicate Page Many Times - This opens a question box where you type how many copies of the pages you want to paste in the same book.

  • Copy Page - This command puts a copy of the page on your computer's clipboard. Then you can open another book in the Edit tab and use the Paste Page command to paste the page in that book (more).

  • Copy Hyperlink - This command puts a URL on your clipboard so you can paste a hyperlink in another page of the same book that will open this page.

  • Paste Page - This command pastes the page you copied to your clipboard in the current book.

  • Remove Page (more).

  • Choose Different Layout (more).
