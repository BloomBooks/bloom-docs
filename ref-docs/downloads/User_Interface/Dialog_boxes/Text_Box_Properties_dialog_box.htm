Text Box Properties dialog box

There is a Change Layout control that allows you choose text box properties. These properties include languages, vertical alignment of words, borders and background, and hints for translators.

Open and use the Text Box Properties dialog box:

  1. Click the end of the Change Layout button to display the controls.

  2. Click the gear button (), the one that is near the center of the text box that you want to change.

The Text Box Properties dialog box appears.

You can use any of these tabs:

Language

  • Select () the language that a text box can use.

Alignment

  • Choose the vertical alignment of the contents in the text box.

Border & Background

  • Choose borders and background options.

Hint Bubbles

  • Work with hint bubbles (Template Starter pages only).

  1. Click the end of the Change Layout control to hide the controls.

Important

  • If you use Add Page, the new page will not have the same properties that you set in this dialog box.

If you use Duplicate Page, the duplicated page will have the same properties as the page you duplicated.

  • The Format dialog box has controls for character and paragraph styles.
