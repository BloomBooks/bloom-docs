Open/Create Collections dialog box

  • To open the Open/Create Collections dialog box, do either of these steps:

    • Click Other Collection on the toolbar.

    • Click the Collections tab.

Then, click the down arrow () that appears next to the collection name and then click Open or Create Another Collection.

A password may be required.

You use this dialog box to

  • Create a new Bloom collection.

  • Open a collection.

  • Browse for a collection on the computer.
