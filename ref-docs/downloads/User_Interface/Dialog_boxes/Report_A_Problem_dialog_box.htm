Report A Problem dialog box

If you experience a problem (bug) with Bloom, please report it.

Prerequisite

  • Your computer must have a connection to the internet.

To submit a problem report, do these steps:

  1. On the (Get Help) menu, click Report a Problem.

The Report A Problem dialog box opens.

  1. See Information for problem reports to learn what to type.

  2. Click and drag the slider control () to indicate how often this problem happens.

  3. Enter your email address, if it is not already shown in the Email box.

  4. Do any of these steps:

    • Select () Include Book if you think it would be helpful to include the book.

    • Leave Include this screenshot selected () (recommended).

    • Click LEARN MORE to see the other information that will be sent automatically with your report (optional).

  5. Click SUBMIT.

Important

  • The report you submit will be visible on the Internet.

If you want your report to remain private, email your report to [email protected].

The Will not be private link informs you of this email address.
