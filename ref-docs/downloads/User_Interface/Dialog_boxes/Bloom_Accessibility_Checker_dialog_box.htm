Bloom Accessibility Checker dialog box

  • To use this dialog box, select () ePUB in the Publish tab. Then, click the Accessibility Checker link (below Settings).

This dialog box has these three tabs:

Learn About Accessibility

In this tab, you can learn about accessibility.

Note that the definition of the ePUB format link opens an internet site (https://en.wikipedia.org/wiki/EPUB).

Accessibility Checklist

In this tab, you see any problems Bloom finds from an accessibility check of your book.
The checklist items include:

  • Audio for all text, Descriptions for all images and Audio for all image descriptions.

(purple ?) appears when audio is missing because there is at least one text box that does not have any text.

(red X) means there are problems you need to correct. The problems are listed. For example, you might need to record audio or add image descriptions.

(green check mark) means there are no problems for the checklist item.

No essential information by color - select () this check box when you are sure there is none. Use the Impairment Visualizer to help verify this.

Ace by Daisy Automated Checks

  • If Ace is installed, you see any problems Ace finds from an accessibility check of the ePUB itself.

It displays results in these four tabs:

Violations, Metadata, Outlines and Images.

To learn about them, review the information at these Internet sites:

    • https://daisy.github.io/ace/

    • https://inclusivepublishing.org/toolbox/accessibility-checker/getting-started/

    • https://daisy.github.io/ace/docs/report-html/.

  • Ace uses external stylesheets so it requires an Internet connection.

  • You can leave this dialog box open as you correct problems and click the Refresh button to update the report.
