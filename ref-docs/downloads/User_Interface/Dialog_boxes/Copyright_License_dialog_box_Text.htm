Copyright and License dialog box - Text

Use this dialog box for the textual content in the book. It is intellectual property. The information appears on the Credits Page, as metadata when you publish to the Web. Also, Creative Commons selections appear in the Book Metadata dialog box.

  1. In the Edit tab, click the Credits Page.

  2. Click  in the callout box.

The Copyright and License dialog box opens.

  1. Click the COPYRIGHT tab.

  2. Select () Not a translation or a version if you should continue to use the same copyright and license as the original book. Then, click OK.

Otherwise, do these steps:

  1. Type the year in the Copyright Year box.

  2. Type the name in the Copyright Holder box.

  3. Click the LICENSE tab

  4. Select () the option that best describes ways the copyright holder allows others to use this book.

    • Anyone can use it however they want, sets PUBLIC DOMAIN.

    • Everyone is allowed to requires you to select () specific options.

The Creative Common symbol shows the result of your selections.

You cannot clear () Copy this book for free.

  • Click the down arrow () and then choose 4.0 or Intergov 3.0.

  • Type in the box to clarify or grant any additional permissions.

  • No one can modify or copy it without explicit permission sets All Rights Reserved.

  • Other sets Custom.

    • Type the specifics of your custom license in the box.

  1. Click OK.

Note

  • The Not a translation or new version check box appears if the book you are editing could or should continue to use the copyright and license as the original book. 

  • Needs Copyright is also a button you can click. It appears when your copyright and license is not complete.

  • About CC0 opens https://creativecommons.org/publicdomain/zero/1.0/.

Other About links appear depending on your selections ().

  • Creative Commons license appears in the Book Metadata dialog box for book accessibility.
