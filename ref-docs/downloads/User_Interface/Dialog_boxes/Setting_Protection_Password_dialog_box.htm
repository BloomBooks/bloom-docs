Settings Protection Password dialog box

Bloom can be set to require a password when someone uses the toolbar. In this case, you will see the Settings Protection Password dialog box.

  1. In the Factory Password box, type the password.

    • Select () Show Characters if you want to see the characters you typed.

Then, you can make sure you typed them correctly.

  1. Click OK.
