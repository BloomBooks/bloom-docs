Edit tab toolbar

This toolbar appears in the Edit tab:

Toolbar buttons:

  • Paste - paste clipboard contents where the cursor is (the insertion point).

  • Cut - cut selected text to the clipboard.

  • Copy - copy selected text to the clipboard.

  • Undo - reverse the most recent change.

  • - This button shows how many languages are selected.

Click it, and then select () or clear languages.

If you use two or three languages, look at all the pages. A red font color or red line means that the text box is too full.

  • - This button shows you the page size and orientation.

Click it, and then click a different page size and orientation (if permitted).

  •  - Click to open and use the Book Settings dialog box to set the maximum size of images in eBooks.

  • - This button shows you the current User Interface (UI) language.

Click it, and then click a language to change the UI language.

  • - Click it to open the (Get Help) menu.

  • - The number shows the zoom percentage.

Click to zoom out; click to zoom in.

Tip

  • Some buttons may be hidden or locked.

  • The tool box can appear vertically on the right side of the Edit tab.

  • The Languages tab is in the Settings dialog box.
