Help menu

(Get Help) on the toolbar has these menu commands:

  •  Help... opens this Bloom Help file.

  •  More Help (Web) opens https://docs.bloomlibrary.org/. 

  • Training Videos opens a list of videos.

  • Building Reader Templates opens a PDF file. It gives an overview about how to make templates you can give to others.

  • Using Reader Templates opens a PDF file. It gives an overview about how to use the templates someone gave you.

  • Ask a Question opens the Language Software Community internet site https://community.software.sil.org/c/bloom.

  • Request a Feature opens the Language Software Community internet site https://community.software.sil.org/c/bloom/featurerequests.

  • Report a Problem opens the Report A Problem dialog box. You can use it type a problem report and submit it to the developers.

You can find the Log.txt file in the Bloom folder that matches your version of Bloom here:
Windows® -  C:\Users\<your name>\AppData\Local\Temp\SIL
Linux - /tmp/SIL/

  • Release Notes opens https://docs.bloomlibrary.org/Release-Notes/.

  • Check For New Version checks an Internet site for a more recent version of Bloom. If there is one, it starts the installer (Windows® only). See Also: Settings.

  • Registration opens the Register Bloom dialog box. Type your name, email address, organization name and a description how you are using Bloom.
    Then, click Register.

  • BloomLibrary.org opens https://bloomlibrary.org/.

  • About Bloom opens a window that lists version numbers and other information about the Bloom program. This information is useful if you need more help.

Tip

  • See https://community.software.sil.org/about to learn about the Language Software Community internet site.
