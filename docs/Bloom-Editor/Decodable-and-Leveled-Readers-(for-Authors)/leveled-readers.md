# Intended audience {#33c4bb19df12800ebcbfdbc957fb5655}


This material is intended for **local authors** who will use [Reader Templates](/glossary#32f4bb19df1280fdb4a3d77da7c693fd) created by a Literacy Specialist ([as described here](/literacyspecialist-DRLR)) to make books that fit within each template's constraints.


# What you should already know {#33c4bb19df12808c8b2ff62ccf1c7c09}


Although Bloom is easy to learn and use, it has several parts. In this documentation, we want to focus on using [**Reader Templates**](/glossary#32f4bb19df1280fdb4a3d77da7c693fd). We assume that you have at least used Bloom a little bit. You should have experimented with making a new book using the “Basic Book” template. You should know how to add new pages and illustrations.


# What you will learn {#33c4bb19df1280aa9b76d60b1f6b7612}


You will learn how to install a “[Reader Template Bloom Pack](/glossary#32f4bb19df12802891c5f7a73c2ed890)” configured for your language. You will then learn how to select a template that fits the kind of book you want to create, and to use the help that Bloom will give you to select words and keep your book at the appropriate level or stage.

# Getting set up in Bloom {#33c4bb19df12808188a7cf11243f7e57}


If you haven’t done so already, get Bloom installed and [make yourself a new Local Language Collection](/create-a-new-collection)_._  Try [making some new books ](/creating-a-new-book)for fun, using the “Basic Book” template. When you feel comfortable with adding pages, adding images, etc., you’re ready to move on to using a Reader Template.
