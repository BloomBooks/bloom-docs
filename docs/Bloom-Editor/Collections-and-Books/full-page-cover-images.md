:::tip

This is a new feature in Bloom 6.2 for clients with a Bloom Subscription.

:::




When you choose the option `Fill the front Cover with a single image`, the front cover image will fill the entire front cover from edge to edge. Since the book’s title (and other information) will no longer be visible, you must add the title to the image itself using a photo editor (see example below).
