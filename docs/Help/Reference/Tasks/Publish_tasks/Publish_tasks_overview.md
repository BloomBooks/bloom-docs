## Publish tab tasks overview

In the ![](/ref-docs-assets/images/User_Interface/Tabs/PublishTab.png) **Publish** [tab](../../User_Interface/Tabs/Tabs_overview.md), you can do the tasks listed below to publish your book whether printed on paper or published [digitally](Digital_publishing_options.md).

|  |  |
| --- | --- |
| ![](/ref-docs-assets/images/Tasks/Publish_tasks/PDFandPrint.png)<br/>**PDF & Print** | [Publish to PDF & Print](PDF_and_Print.md)<br/>• [Multiple color profiles](Multiple_color_profiles_for_PDF_output.md)<br/>• [Print a PDF file](Print.md)<br/>• [Save PDF file](Save_PDF_file.md) |
| ![](/ref-docs-assets/images/Tasks/Publish_tasks/UploadNew.png)<br/>**Web** | • [Publish to Web](Publish%20to%20Web.md)<br/>• [Sign up for BloomLibrary.org](Sign_up_for_BloomLibrary.md)<br/>• [About Book Features](Features.md)<br/>• [Add custom thumbnail for Bloom Library](Add_custom_thumbnail_for_BloomLibrary.md) |
| ![](/ref-docs-assets/images/Tasks/Publish_tasks/Android_Button_New.png)<br/>**BloomPUB** | **Publish as BloomPUB**<br/>• [Making BloomPUB Apps with Reader App Builder](Making_BloomPUB_Apps_with_Reading_App_Builder.md)<br/>• [Viewing BloomPUBs on Bloom Reader (Android)](Make_a_BloomPUB_file_overview.md)<br/>• [Viewing BloomPUBs on Windows](Viewing_BloomPUBs_on_Windows.md) |
| ![](/ref-docs-assets/images/Tasks/Publish_tasks/VideoButton.png)<br/>**Audio & Video** | [Publish as Audio or Video](Create_audio_or_video_of_book.md) |
| ![](/ref-docs-assets/images/Tasks/Publish_tasks/EPUB_buttonSmall.png)<br/>ePUB | • [Publish as ePUB](Make_an_ePUB_book_overview.md)<br/>[Save ePUB file](Save_EPUB_file.md)<br/>• [Accessibility](Accessibility.md)<br/>• [Use the Bloom Accessibility Checker](Use_Bloom_Accessibility_Checker_dialog_box.md)<br/>• [Use the Book Metadata dialog box](Use_the_Book_Metadata_dialog_box.md) |
